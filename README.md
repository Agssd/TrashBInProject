# ЭкоБаллы

**Краткое описание:** Android-приложение (Kotlin) для геймификации экологии — пользователи получают баллы за фото полных мусорных баков.

## Основное
- **Авторизация** пользователей
- **CameraX** для фото
- **ML-модель** (обучена на 2к фото) для верификации
- **Room DB** для локального хранения
- **Jetpack Compose** UI, MVVM архитектура
